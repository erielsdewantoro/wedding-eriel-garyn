 Tambah Satu Tamu

Paste langsung nama-nama dari chat WhatsApp, notes, atau ketik manual. Satu nama per baris.

✅ Otomatis bersihkan nomor urut, tanda baca, spasi berlebih.
✅ Nama duplikat & yang sudah ada di database ditandai otomatis.

Buka file Excel/CSV kamu, select kolom nama → copy (Ctrl+C) → paste di sini.

Cara dari Excel:
1. Buka file Excel di laptop
2. Klik header kolom nama (misal kolom A)
3. Ctrl+C (copy)
4. Paste di kotak di bawah

Cara dari Google Sheets:
Sama — select kolom → Ctrl+C → paste di bawah
⚠️ Customer kirim nama via foto?
Ini yang paling umum terjadi. Solusinya:

Opsi 1 — Google Lens (gratis, cepat):
Buka foto di HP → tahan gambar → "Cari dengan Google Lens" → copy teks yang terdeteksi → paste di tab WA di sebelah.

Opsi 2 — Screenshot + Word:
Buka Word → Insert → Pictures → klik kanan → "Copy Text from Picture" → paste ke tab WA.

Opsi 3 — Ketik manual di bawah:
Kalau foto tidak terbaca, ketik langsung satu per satu di sini.
